## Описание программы
Movie Library — это консольное приложение для ведения своей личной кинетеки. Позволяет
добавлять записи о названии фильма, годе выпуска, жанре, оценке, фильтровать их по жанру и году."*
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`
---
## Как запустить
```bash
git clone https://github.com/твой-логин/weather-diary.git
cd weather-diary
python main.py
Как пользоваться
1. При запуске появляется меню:
2. text
1 — Добавить запись
2 — Показать все записи
3 — Фильтровать по дате
3. 4 — Выйти
4. Выбери нужный пункт, введи данные.
5. Данные автоматически сохраняются в data.json.
Пример работы
Добавление записи:
Введите дату (ГГГГ-ММ-ДД): 2026-05-15
Введите температуру: 22.5
Введите описание: Солнечно
Осадки (да/нет): нет
✅ Запись добавлена!
Фильтрация по температуре > 20°C:
2026-05-15 | 22.5°C | Солнечно | Нет осадков
2026-05-14 | 21.0°C | Облачно | Нет осадков
