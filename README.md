## Описание программы
Movie Library — это консольное приложение для ведения своей личной кинетеки. Позволяет
добавлять записи о названии фильма, годе выпуска, жанре, оценке, фильтровать их по жанру и году."*
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`
---

```bash
git clone https://github.com/moriki799/Movie Library.git
